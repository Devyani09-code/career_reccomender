# career_reccomender
career_recommender_system
The project is an evaluation system that helps you decide your stream.The platform is built using Machine Learning, Psychometric Analysis, and Streamlit.
Our project helps students discover a career path based on academic performance,personal interests,psychometric analysis,AI_based learning models and market demands.

The Machine Learning models used are:
Scikit-learn
Logistic Regression
Naive Bayes

The data was read and analysed using pandas,numpy library.
The visualisation of scores and analysis was prepared using matplotlib library.
The psychometric Asssesment uses RIASEC personality components,personal interests, preferences and behavorial analysis to evaluate and deliver the results.

The score given is based on:
Academic contribution
Interest match percentage
Personality analysis
AI confidence score

The technologies used for this project are:
HTML
CSS
Streamlit
Python
the record of users was saved using supabase
